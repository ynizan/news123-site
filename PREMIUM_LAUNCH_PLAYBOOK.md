# Premium Launch Playbook

## Phase 1: Collect Waitlist (Current)

### Daily Process
1. Check GitHub Issues with label `waitlist`
2. Add to tracking spreadsheet:
   - Email, Name, Company, Plan interest, Signup date, Source, Feature

### Weekly Report
1. Count signups by plan
2. Track which features drive most interest (Alerts, CSV, Historic data, API)
3. Note any enterprise inquiries for immediate follow-up

## Phase 2: Pre-Launch (2 weeks before)

### Setup Stripe
1. Create Products: Pro ($39/mo, $390/yr), Business ($99/mo, $990/yr)
2. Get Payment Link URLs
3. Test checkout flow

### Update Website
1. Replace `/coming-soon/` links with Stripe URLs
2. Keep "Talk to Us" for Enterprise
3. Deploy

### Email Waitlist
**Subject:** News123 Pro is LIVE - Your Early Access

Offer: 20% off first 3 months with code EARLYBIRD

## Phase 3: Launch Day

### Enable Features
1. ✅ Alert subscriptions functional
2. ✅ CSV export links active
3. ✅ Historic data visible (if ready)
4. ✅ API docs published (if ready)

### Monitor
- Plausible: Conversion rates
- Stripe: Successful payments
- Email: Customer support inquiries

## Metrics to Track

### Acquisition
- Waitlist → Paid conversion rate (target: 20-30%)
- Source of signups
- Plan distribution (Pro vs Business ratio)

### Activation
- % who set up first alert (target: >50% within 7 days)
- % who export first CSV (target: >30% within 7 days)

### Retention
- Monthly churn rate (target: <5%)
- Support ticket volume

### Revenue
- MRR (Monthly Recurring Revenue)
- Annual vs monthly split (target: 30% annual)
- Upgrade rate (Pro → Business, target: 10%/year)
